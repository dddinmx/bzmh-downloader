# 📖 使用方法

#### 📝 步骤

#### 1. 克隆本仓库

```
git clone https://github.com/dddinmx/bzmh-downloader.git
```

#### 2.安装依赖

```
pip3 install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

#### 3.运行
```
python3 bzmh-downloader.py   (为了下载稳定，请关闭代理软件使用!)
```
