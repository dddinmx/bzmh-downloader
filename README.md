# bzmh-downloader
下载包子漫画内容。仅学习研究使用。  
开发环境: Python 3.12.4

### 安装使用  
``安装方法: git clone https://github.com/dddinmx/bzmh-downloader.git``  
``pip3 install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/``

### 功能  
- [x] 断点下载，下载中断退出后，重新运行选择[更新]可续下。  
- [x] 保存为PDF、CBZ格式
